## GAMEPLAY KEYS
---
* A move left
* D move right
* W climb
* S hide
* E dash
* Q glide
* SPACE jump
* ESC open settings menu


### DEBUG FUNCTION KEYS
---

* 1 Set player at the start of map 1
* 2 Set player at the start of map 2
* 3 Set player at the start of map 3
* F1 Visualize the colliders / logic
